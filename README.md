# README

- 推上 Heroku

  - 申請 Heroku 帳號

    目前申請 Heroku 帳號是免費的，只要填寫一些簡單的基本資料即可申請。

  - 安裝 Heroku Cli

  - 終端機登入 Heroku

    ```
    $ heroku login
    Enter your Heroku credentials.
    Email: xxx@xxx.com.tw
    Password (typing will be hidden):
    Logged in as xxx@xxx.com.tw
    ```

  - 部署伺服器

    ```
    $ heroku create
    ```

  - 推向 Heroku

    ```
    $ git push heroku master
    ```
